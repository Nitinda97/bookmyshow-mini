# BookMyShow Mini Challenge Starter

This is a starter Spring Boot project (v3.5.7, Java 21) structured for Codecrafters-style, step-by-step backend engineering challenges. Clone and start coding for Stage 1!

## Structure

- src/main/java/com/bookmyshowmini/
    - Application.java (SpringBoot entry)
    - controller/
    - service/
    - repository/
    - model/
    - dto/
- src/main/resources/
    - application.properties
- pom.xml (Java 21, Spring Boot 3.5.7, minimal dependencies)

## Usage
Clone, open in IntelliJ/Eclipse and start coding.

## Stage Progression
Stage 0: Repo setup (you're here!)
Stage 1: Start with basic features (TCP binding, then REST, as per challenge stages).

More stages to come!
